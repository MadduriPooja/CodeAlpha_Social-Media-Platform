# Mini Social Media App

A simple social media application built with **HTML, CSS, JavaScript (Frontend)** and **Express.js (Backend)**.  
This project demonstrates core social features like user profiles, posts, comments, likes, and a follow system — all using an in‑memory data store (no external database).

---

## 🚀 Features
- User profiles (create users with username & bio)
- Posts (create, view, like)
- Comments (add comments to posts)
- Like system (like/unlike posts)
- Follow system (follow/unfollow users)

---

## 🔎 How to View the Site

Follow these steps to run and view the app locally:

1. Clone the Repository
    After cloning..

2.Make sure you have Node.js installed. Then run:
    npm install express

3.Go into the backend folder and start the server:
    cd backend
    node server.js

4.You should see:
    Server running on port 5000

5.Now open your browser and go to:
    http://localhost:5000

This will load the frontend (index.html) served by Express.
